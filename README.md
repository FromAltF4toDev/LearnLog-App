# LearnLog App

Track your learning progress with this simple single page web application.


## Features

-Topic creation
-Session creation and tracking
-Task Managment with deadlines
-Reminder system
-Dashboard statistics
-Notes and links per topic





## Steps to do

-Create a learning topic
-Start a session timer to create a new session
-add tasks with reminder and expiration date
-add notes and links
- track your progress trough the dashboard



## Used language modules

-ECMAScript (Javascript)
-CSS
-HTML

## How to run

-Download repository
-Open index.html in browser

## How to reset localStorage

WARNING: the following step will reset your whole localStorage data.

-open the developer tools in your browser
-switch to console
-enter : localStorage.removeItem("state") and run the command
-you should see undefinded
-reset the website trough reset button from browser
